# 仪表板操作
---

**[预览操作]**

**仪表板**资源路径是有关每个 Power BI 仪表板的属性集合。

Power BI REST API 具有以下**仪表板**操作：

- [获取仪表板](Get-Dashboards.md)：返回所有**仪表板**对象的 JSON 列表。
- [获取磁贴](Get-Tiles.md)：返回所有**磁贴**对象的 JSON 列表。
